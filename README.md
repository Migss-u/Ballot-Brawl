# Ballot-Brawl
One Button android game built with Unity & C#

## Features
- **Collect votes/points**: Gather as many points as possible within the time limit to become the top player.  
- **Power-ups**: Pick up power-ups on the map to gain advantages over AI opponents.  
- **Avoid enemies**: Dodge protestors or enemy characters to prevent losing health.
